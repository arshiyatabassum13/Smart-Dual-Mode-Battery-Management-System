# Smart Dual-Mode Battery Management System for Electric Vehicles

A smart battery management system designed for electric vehicles that dynamically configures battery cells based on operating conditions. The system improves charging efficiency, enhances battery safety, and supports reliable electric mobility applications.

## Features

- Intelligent battery cell reconfiguration
- Improved charging efficiency
- Enhanced battery safety
- Compact and low-cost design
- Suitable for electric vehicle applications

## Technologies Used

- MATLAB Simulink
- Embedded Systems
- Relay Switching
- Battery Management Systems
- Electric Vehicle Technology

## Applications

- Electric Two-Wheelers
- Smart Mobility Solutions
- EV Research and Innovation
